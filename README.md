# Student Grade Manager (Python)
A simple beginner Python project for storing student names and grades, calculating average grades, highest grade, and showing the top performer.

## Features
- Stores student names in a list
- Stores student grades in a list
- Calculates the class average
- Finds the highest grade
- Shows the top student
- Prints all students with their grades

## How to Run
- Open the project in Python
- Run: python student_grade_manager.py
